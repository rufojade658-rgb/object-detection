# 🎥 Live Object Detection & Tracing using YOLOv8 + Streamlit

A real-time object detection web application built with Streamlit, WebRTC, and YOLOv8.  
This project uses your webcam to detect and track objects live in the browser.

---

## 🚀 Features

- 📷 Real-time webcam streaming
- 🧠 YOLOv8 object detection
- 🎯 Object tracking with bounding boxes
- ⚡ Fast live processing
- 🌐 Browser-based interface using Streamlit

---

## 🛠️ Technologies Used

- Python
- Streamlit
- streamlit-webrtc
- OpenCV
- Ultralytics YOLOv8
- AV
